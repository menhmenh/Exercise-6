# Data Visualization Exercise 6.1

## Project Structure

- **CSS/**: Stylesheets for the visualization
  - `base.css`: Base styling
  - `visualisations.css`: Visualization-specific styles

- **js/**: JavaScript modules
  - `shared-constants.js`: Shared constants
  - `load-data.js`: Data loading functionality
  - `histogram.js`: Histogram visualization
  - `interactions.js`: User interactions handling

- **data/**: Data files
  - `Ex6_TVdata.csv`: TV ratings data

- **index.html**: Main HTML file
